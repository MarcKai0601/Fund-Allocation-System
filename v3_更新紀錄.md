# V3 更新紀錄

**版本**: v3.0.0  
**日期**: 2026-02-27

## ✅ RWD 響應式設計（手機友善）

### Sidebar 導航

- **手機版**（< 768px）：固定頂部導航列 + 漢堡選單按鈕
  - 點擊漢堡按鈕開啟左側滑出式抽屜
  - 半透明遮罩背景，點擊可關閉
  - 切換頁面自動關閉抽屜
  - 開啟時鎖定背景滾動
- **桌面版**（≥ 768px）：維持原有固定側邊欄

### Layout 佈局

- 內容區 padding：手機 `16px`、桌面 `32px`
- 手機版自動預留頂部導航列高度，內容不被遮擋

### 總覽 Dashboard

- 統計卡片：手機單欄 → 平板雙欄 → 桌面五欄
- 頁面標題在手機端自動堆疊排列
- 「上次更新」時間在手機端隱藏（節省空間）

### 資金管理

- 統計卡片：手機單欄 → 平板以上雙欄
- 操作按鈕在手機端自動堆疊

### 交易紀錄

- 新增交易 Dialog：手機端限制最大高度 90vh，內容可滾動
- 交易表格：手機端橫向捲動，保持完整欄位不破版
- 頁面標題與按鈕在手機端堆疊排列
