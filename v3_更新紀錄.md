## ✅ RWD 響應式設計（手機友善）

### Sidebar 導航

- **手機版**（< 768px）：固定頂部導航列 + 漢堡選單按鈕
  - 點擊漢堡按鈕開啟左側滑出式抽屜
  - 半透明遮罩背景，點擊可關閉
  - 切換頁面自動關閉抽屜
  - 開啟時鎖定背景滾動
- **桌面版**（≥ 768px）：維持原有固定側邊欄

### Layout 佈局

- 內容區 padding：手機 `16px`、桌面 `32px`
- 手機版自動預留頂部導航列高度，內容不被遮擋

### 總覽 Dashboard

- 統計卡片：手機單欄 → 平板雙欄 → 桌面五欄
- 頁面標題在手機端自動堆疊排列
- 「上次更新」時間在手機端隱藏（節省空間）

### 資金管理

- 統計卡片：手機單欄 → 平板以上雙欄
- 操作按鈕在手機端自動堆疊

### 交易紀錄

- 新增交易 Dialog：手機端限制最大高度 90vh，內容可滾動
- 交易表格：手機端橫向捲動，保持完整欄位不破版
- 頁面標題與按鈕在手機端堆疊排列

## 🎨 手機端美觀優化

### 字體縮放

- 手機端 (< 768px) 全域字體自動縮小 2px，避免文字溢出
  - `A` 模式：16px → 14px
  - `A+` 模式：18px → 15px
  - `A++` 模式：20px → 16px

### 持股庫存（手機版卡片排列）

- 手機端不再使用 8 欄表格（容易擠在一起）
- 改為**卡片式排列**，每支股票一張卡片：
  - 頂部：股票代號 + 名稱 + 報酬率 Badge
  - 下方：6 項數據以 2×3 格線排列（庫存、現價、均價、漲跌、市值、損益）
- 桌面版維持原有表格顯示

### 統計卡片

- 手機端 padding 縮小、數值字體縮小（`text-base`）
- 桌面端維持原有大小（`text-lg` / `text-xl`）
- 圖示在手機端略小
