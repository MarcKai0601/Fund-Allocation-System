# V4.1 修改紀錄 — Java MGR SSO 整合

**版本**: v4.1.0  
**日期**: 2026-03-05  
**類型**: Security Upgrade — 對接外部 Java MGR SSO 系統

---

## 變更背景

Java MGR (SSO 系統) 已實作完成，登入成功後將 Session 存入與 Python 共用的 Redis：
- **Key**: `token:<token>`
- **Value**: `{"UserId": 1, "Roles": {"FAS": ["ADMIN"], "MGR": ["USER"]}}`

本次升級讓 FAS 後端從 Redis 讀取此 Session，實現跨系統身分驗證與 RBAC 權限控制。

---

## 後端修改

### `app/core/security.py` — 全面重寫

| 變更 | 說明 |
|------|------|
| `HTTPBearer` | 改用 FastAPI 內建 `HTTPBearer` 自動解析 `Authorization: Bearer <token>` |
| `get_current_user` | 從 Redis 查 `token:<token>`，解析 JSON Session，回傳 `{"UserId", "Roles"}` dict |
| `require_fas_access` | 新增 RBAC，檢查 `Roles` 中是否有 `"FAS"` Key，回傳 `UserId: int` |
| `get_valid_portfolio` | 鏈式依賴 `require_fas_access`，驗證 Portfolio 所有權 |

### `app/core/redis_client.py` — Token 格式對齊 Java MGR

| 變更 | Before | After |
|------|--------|-------|
| Token Key | `auth:token:<token>` | `token:<token>` |
| Token Value | plain string (`user_id`) | JSON `{"UserId": int, "Roles": dict}` |
| 新增函式 | — | `get_auth_session()` |
| `set_auth_token()` | `(token, user_id, ttl)` | `(token, user_id, roles, ttl)` |

### `app/api/portfolios.py` — 套用 RBAC

- 所有 API 依賴由 `get_current_user` → `require_fas_access`
- `user_id` 型別由 `str` → `int`

### `app/api/dev_auth.py` — 模擬登入更新

- `user_id` 改為 `int` (預設 `1`)
- 新增 `roles` 參數 (預設 `{"FAS": ["ADMIN"]}`)
- 寫入 Redis 格式與 Java MGR 一致

---

## 前端修改

### `lib/api.ts`

- `devApi.login()` 參數改為 `userId: number` (預設 `1`)
- 回傳型別加入 `roles: Record<string, string[]>`
- Request/Response Interceptors 無需修改 (已到位)

---

## 驗證流程

```
前端 → Authorization: Bearer <token>
         │
         ▼ HTTPBearer 解析
         │
         ▼ Redis GET token:<token>
         │
         ├── 不存在 → 401 Unauthorized
         │
         ▼ JSON {"UserId": 1, "Roles": {"FAS": ["ADMIN"]}}
         │
         ▼ require_fas_access: Roles 有 "FAS"?
         │
         ├── 無 → 403 Forbidden
         │
         ▼ 回傳 UserId → get_valid_portfolio → 操作資料 ✅
```

---

## 影響範圍

- 所有需要驗證的 API 現在會走 `require_fas_access`
- 舊 Token (plain string) 已不相容，需使用新格式
- 開發測試用 `/api/dev/login` 已同步更新
